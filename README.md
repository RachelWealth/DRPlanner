# DRPlanner
> TO DO list app which can display the daily, monthly, yearly plan in one page and use praogress bar to demonstrate your progress 

## Technique stack:
### Backend:
- Node.js
- Express
- MongoDB
- JWT

### Frontend:
- TypeScript
- Sanity
- Tailwind
- React
- Next.js
- Redux

### Deployment:
AWS: amplify, EC2
